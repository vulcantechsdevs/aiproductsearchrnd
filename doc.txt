1.To start backend : open terminal and run the following command 
python -m uvicorn backend:app --reload --port 8000

2.To view the output, normally run the UI.html page. (open with live server option in vs code)

3. before starting backend, make sure 
   1.you have postgre sql installed and have medword db with data. 
   2.you have python 3.11 installed 
   3.you have chroma db installed.
   pip install chromadb   
   pip install uvicorn    
   pip install fastapi uvicorn chromadb sentence-transformers psycopg2-binary
